# data/ フォルダについて

## Excelファイルの配置

このフォルダに受講者数データの Excel ファイルを保存してください。

アプリのサイドバーから直接アップロードして使用することもできます。

## Excelファイルの形式

### シート構成
- 1ファイルに複数シートを作成してください
- **シート名に年度を含めてください**（例：`2025年度`、`2024年度`）
- シート名中の4桁数字が年度として自動認識されます

### 列構成（1行目がヘッダー行）

| 列 | 項目名 |
|----|--------|
| A | 対象学年 |
| B | 開講教室 |
| C | 講座名 |
| D | 授業時間 |
| E | 担当講師 |
| F | 4月度受講人数 |
| G | 5月度受講人数 |
| H | 6月度受講人数 |
| I | 7月度受講人数 |
| J | 8月度受講人数（夏期講習会） |
| K | 9月度受講人数 |
| L | 10月度受講人数 |
| M | 11月度受講人数 |
| N | 12月度受講人数 |
| O | 冬期講習会受講人数 |
| P | 1月度受講人数 |
| Q | 2月度受講人数 |

### 注意事項
- 受講人数が空欄の場合は 0 として扱います
- 空行は自動的に除外されます
- 講座名が空の行は除外されます

## course_aliases.json

講座名の表記ゆれ修正マッピングを保存するファイルです。

アプリの「⚙️ データ管理」タブから編集・保存できます。

```json
{
  "英語特訓（上）": "英語特訓(上)",
  "数学基礎ｺｰｽ": "数学基礎コース"
}
```

このファイルを直接編集することも可能です。
