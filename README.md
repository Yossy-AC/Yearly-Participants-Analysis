# 年次受講者数分析アプリ

塾・教育機関の年次受講者数データを可視化・分析する Streamlit Web アプリです。

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

ブラウザで http://localhost:8501 にアクセスしてください。

## 使い方

1. サイドバーの「Excelファイルをアップロード」からデータファイルを選択
2. サイドバーのフィルター（年度・学年・教室・時間・講座ジャンル・科目・担当講師）で対象データを絞り込み
3. 各タブで分析結果を確認・ダウンロード

## Excelファイルの形式

- 1ファイル・複数シート構成（シート名 = 年度、例：`2025年度`）
- 列構成（左から順）：対象学年、開講教室、講座名、授業時間、担当講師、4月〜2月の受講人数

詳細は [data/README.md](data/README.md) を参照してください。

## タブ構成

| # | タブ | 概要 |
|---|------|------|
| 1 | 📊 メイン | KPI（4項目）+ サマリーグラフ（2×2グリッド：年度別合計・月別推移・学年別分布・教室別分布） |
| 2 | 📈 経年推移 | 年度間の合計推移グラフ＆テーブル（折れ線＋棒グラフ） |
| 3 | 🗓️ 年間推移 | 月別推移グラフ＆テーブル（季節性、年度重ね合わせ） |
| 4 | 🎓 学年別 | 学年別 KPI・年度比較棒グラフ・前年比増減テーブル |
| 5 | 🏫 教室別 | 教室別 KPI・年度比較棒グラフ・前年比増減テーブル |
| 6 | 📚 講座別 | 講座別 KPI・ドリルダウン年次比較・前年比増減テーブル |
| 7 | 👩‍🏫 担当別 | 講師別 KPI・担当人数棒グラフ・ヒートマップ・前年比増減テーブル |
| 8 | 📥 データエクスポート | クロス集計表（学年×年度、教室×年度、講座×年度、担当×年度、月別×年度）・CSV ダウンロード |
| 9 | 🤖 AI分析 | Claude API によるデータサマリー自動生成（全体傾向・増減要因・次年度予測） |

## フィルター

サイドバーで以下の項目を複数選択できます。全タブにリアルタイム反映されます。

- **年度**: 複数年度選択可
- **学年**: 複数学年選択可
- **教室**: 複数教室選択可
- **時間**: 複数時間帯選択可
- **講座ジャンル**: 複数ジャンル選択可
- **科目**: 複数科目選択可
- **担当講師**: 複数講師選択可

各フィルターに「全選択」ボタンがあり、一括で全項目を選択できます。

## AI分析機能

タブ9「🤖 AI分析」では、現在のフィルター条件のデータをもとに Claude がサマリーを自動生成します。

### セットアップ

`.streamlit/secrets.toml` に Anthropic API キーを設定してください:

```toml
ANTHROPIC_API_KEY = "sk-ant-xxxxxxx"
```

> `.streamlit/secrets.toml` は `.gitignore` により追跡対象外です。リポジトリにコミットしないでください。

### 生成内容

1. **全体サマリー**: 受講者数の全体的な傾向
2. **増減の要因分析**: 前年比で大きく変動した項目の考察
3. **次年度の予測**: トレンドをもとにした来年度の見通し
